Why this site

There are a lot of resources on the Internet to whom you would like to come back again and again. To make this possible you need to store somewhere links (bookmarks) to such resources. The paper on which you have written down a link will be lost immediately. Use of the browser to store links is uncomfortable. Everytime search for the desired link using of Google is tedious. So what to do?

Use this site!

The site makes it easy to create your own web pages with links (bookmarks) to your favorite Internet resources. In addition you can place on a page the headlines that are published by the most prominent news agencies of the world. News will be automatically updated with intervals set by you. Then just make your Links And News page in the browser a default page and every time you open the browser your favorite links and news are right in front of to you!

If desired you can publish your Links And News pages and they are available to any Internet user. You can also include pages published by others into the list of your pages and use them as well as your own. The site also provides an opportunity to read news without the use of a specially created for this page.

To create your own Links And News pages it did not require any special knowledge.

Let's get started

How to create a very first page

Let's create your first Links And News Page. To create your own pages, you need to be a registered user. So if you have not registered yet, please register . If you do not want to register - it does not matter, just use pages published by other users or read news .

  1. Navigate to "My pages" in the main menu. Your pages list is empty at the moment.
  2. Choose "My pages" -> "Add". A window will open.
  3. Enter "My page" into "Page name" field.
  4. Enter "Google" into "Link name" field.
  5. Enter "google.com" into "Link address".
  6. Click "Save" button. The window will close.
  7. Congratulations! Your first Links And News page is in front of you. By using the link "Google" you can go to google.com site.
  8. To quickly add to your page links (bookmarks) directly from the Internet page on which you are install Links And News extension for Google Chrome browser and follow the instructions in Bookmark in two clicks section.
Page structure

A page consists of its own properties, such as name and title and may also include columns, rows and links (bookmarks). Columns and rows can be hierarchically nested allowing creating pages of any complexity. Pages, columns and links can include images and descriptions which can be hidden or visible.

Each page has a unique Interenet address that ends with your user name and a page name. For example, if your login is "mylogin" and a name of the page is "mypage" the address of the page will be https://linksnews.org/pages/mylogin/mypage. If you publish your page it will be available at this address to any Internet user. The page will be displayed in the categories you specify.

In order to put links or news on a page you need to add at least one column to the page first. A page can contain up to twelve columns. You can specify a title and a description for each column as well as upload an image. A column can contain links, news or rows. In the simplest case you create a column and add links to it or specify a source of the news. If you want to create a more complex column containing for example links and news you should add two rows to a column first. Then you add a column with links to one row and a column with news to other row. As well as a page a row can contain up to twelve columns.

Page creation

You can create a page in few simple steps.

  1. If you have not yet logged in please log in. To do this choose "Account" -> "Log in" from the menu.
  2. Navigate to "My pages". There are two ways to add a page.
  3. Way 1: Select "My page" -> "Edit". This will open an "Edit pages list" window. Press "Add." A "New page" will appear at the bottom of the window. Click "Save". The window "Edit pages list" will close. The list of pages will display a newly created page.
    Way 2: To quickly create a simple page you can select "My page" -> "Add". This will create a page with one column and one link.
  4. Go to the created page. To do this, click on the page title. If you used the quick page creating you are already there.
  5. Navigate to "Page" -> "Edit". The page will go to edit mode. A next step is to add columns to the page.
  6. Click the edit icon on the right side of a page or click "page" on the left. "Edit page" window appears. Fill required fields, upload an image if desired. To add one or more columns and adjust their width use the buttons at the bottom of the window. In this step add one column and make it the maximum width. Click "Save". The window "Edit page" will close and the page will display the added column. In the next step we will edit the properties of the column and add links there.
  7. Click the edit icon on the right side of a column or click "column" on the left. "Edit column" window appears. By default the column type is "Links". Do not change it. Select a "View mode", "List" or "Tile". If desired you can upload an image and specify a title and description. Then add one or more links with the "Add" button. Click "Save". The window closes and the column displays links added. Next specify the properties of links.
  8. Click the edit icon on the right side of a link or click "link" on the left. "Edit link" window appears. Specify the title for the link, which will be displayed on the page. For example, if you create a link to the Google website type in "Google". Set the address of the link. The easiest way to do this is to go to the website on the Internet for which you are creating the link and copy the adderss for the site from the browser's address bar, and then paste the copied URL into the "Address" field. In this case it is "http://google.com". Click "Save". The window closes and a link title will be displayed inside the link edited.
  9. To complete the creation of the page choose "Page" -> "Leave edit mode". Page will leave edit mode and enter browse mode. Now you can use the created links to go to the corresponding Internet pages.
Bookmark in two clicks

Adding a link (bookmark) using the method described in section Page creation may seem too complicated. To quickly bookmark a page directly from this page use Links And News extension for Google Chrome browser.

  1. Use Google Chrome browser.
  2. Install Links And News extension .
  3. Navigate to an Internet page you want to bookmark. Press icon on the right of the browser's address bar. A popup window Links And News will open.
  4. If you are not authorized on Links And News website an authorization form will be shown. Enter your login and password and press "Login". After successful authorization a form for bookmarking opens.
  5. Select a page in the "Page" drop-down list. To make a page appear in this list the page should have at least one column of type "Links". To create a page follow the instructions in sections Let's get started or Page creation.
  6. Select a column in drop-down list "Column".
  7. Fields "Link Name" and "Link Address (Url)" will be filled in automatically. You can change them if you wish.
  8. Press "Add link to Links And News" button. A link will be added to the links list in a column specified.
  9. Recent page and column you selected will be set by default any time you open this popup window.
News on page

To place news on your page it is necessary to add there a column, specify the column type "News" and choose a news source from a list of available news sources.

  1. Navigate to "My Page". Select the page on which you want to add news or create a new page. Go to this page.
  2. Switch to edit mode using "Page" -> "Edit".
  3. Select the existing column or add a new one. Click the edit icon on the right side of the column. "Edit column" window will open.
  4. Select "News" from the "Column type" drop-down list.
  5. Click on "Add source" button. A window "News sources" will open showing a list of available news sources.
  6. Select a source. To do this click on the round button on the left for the appropriate source.
  7. Click "Save". Window "News sources" will close. The name of the selected news source appears in the the "Edit column" window.
  8. Click "Save". The window "Edit column" will close.
  9. To complete the addition of news on the page select "Page" -> "Leave edit mode". Page will go to browse mode. News from the source you have specified will be displayed in the appropriate column.
Links and news in one column

You can place links and news or any their combination within the same column. To do this it is necessary to specify column type "Rows" and add the number of rows. Then add a column inside each row with news or links as well as you do that for a page.

  1. Navigate to "My Pages". Select the page on which you want to add news or create a new one. Go to this page.
  2. Select the existing column or add a new one. Click the edit icon on the right side of the column. "Edit column" window will open.
  3. Select "Rows" from the "Column type" drop-down list.
  4. Add the necessary number of rows.
    5. Click "Save". The window "Edit column" will close. Added rows are displayed in the column. Further you need to add columns containing links or news to the rows.
  5. Click the edit icon on the right side of a row or click "row" on the left. "Edit row" window will open. Add one or more columns.
  6. Click "Save". The window "Edit row" will close. Added columns appear within the row.
  7. To add links or news to a column use the instructions in "Page creation" and "News on page" sections of this help.

FAQ

Q: How do I bookmark a page directly from that page?
A: Use Google Chrome Browser. Install Links And News extension . Follow the instructions in Bookmark in two clicks section.

Q: How to restore a forgotten password?
A: Send us a message. We will send your new password to your email address.

Q: How do I add a page published by others to "My Pages"?
A: Go to the page you're interested in and select "Page" -> "Add to My pages"

Q: How to copy a page to "My Pages"?
A: Go to the page you're interested in and select "Page" -> "Copy to My pages". You can copy any page including your own.

Q: What is the difference between "Add to My pages" and "Copy to My pages"?
A: "Add to My pages" means to include a page published by someone in a list of your pages. You cannot edit this page but you can remove it from the list of your pages. If the page's author updates the page changes will be visible to you. And "Copy to My pages" means to make a copy of the page and put it in the list your pages. You can use the copy the same way as you use your other own pages.

Q: How do I delete a page?
A: Select "My pages" -> "Edit". An "Edit pages list" window will appear. Select the desired page and click "Delete." Then save the changes.

Q: How to switch site language?
A: If you are not logged in choose "Language" -> <language>. If you are logged in navigate to "<Your login>" -> "My account", choose a language from the drop-down list "Language" and save changes by clicking on "Save" button located at the form bottom.
